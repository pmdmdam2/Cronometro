# Cronometro

Ejemplo en el que se muestra el funcionamiento básico del widget Chronometer de Android. La interfaz está compuesta por:
- 5 Botones: inicio, parada, reinicio, reset automático cada 10 segundos y cuenta atrás
- 1 Cronómetro
